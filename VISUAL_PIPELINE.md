## Build Commands Summary

```bash
# Stage 1: Test Python
python3 hello_world.py

# Stage 2: Compile Cython to C
cython hello_world.pyx --embed

# Stage 3: C is ready (hello_world_wasm.c)

# Stage 4: Compile to WebAssembly (requires Emscripten)
emcc hello_world_wasm.c -o hello_world.js \
  -s EXPORTED_FUNCTIONS='["_create_hello_world"]' \
  -s EXPORTED_RUNTIME_METHODS='["ccall","cwrap"]' \
  -s MODULARIZE=1 -O3

# Stage 5: Build TypeScript
cd typescript && npm install && npm run build
```
